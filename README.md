#P Y-OpenEMM
### A set of functions to automate your OpenEMM 2015 R3 application.

#### Required libraries:

- os
- time
- selenium

#### Other requirements:

- Any selenium driver (chrome recommended)

------------


##  Functions (in development):

### Login
 `login()`


**Possible arguments:**

`address='domain'`
`password='password'`

**Example:**
`login(address='thic-cat.com', password='lazycat')`

=========================================================

### Password change
`change_password()`

**Possible arguments:**

`old='oldpasswd'`
`new='newpasswd'`

**Example:**
`change_password(old='lazycat', new='baldcat')`


=========================================================
